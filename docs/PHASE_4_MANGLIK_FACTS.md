# Phase 4 Manglik/Kuja calculation facts

Status: raw placement facts only. These factors are separate from the 36-point Ashtakoota total and are not marriage rejection rules.

## Frozen convention

Profile: `lagna_moon_venus_1247812_v1`

For each chart, Mars is counted independently from:

1. Lagna
2. Moon sign
3. Venus sign

A reference-point fact is flagged when Mars occupies whole-sign house 1, 2, 4, 7, 8, or 12 from that reference point.

## Response behavior

Each chart returns exactly three factors. Every factor contains:

- reference point
- calculated Mars house
- flagged boolean
- stable rule ID
- convention notes

## Excluded behavior

This version does not apply:

- cancellation rules
- severity rankings
- combined-couple cancellation
- pass/fail decisions
- relationship-outcome claims
- medical, fertility, lifespan, wealth, or event-timing claims
- remedies or remedy upselling

The house set and Lagna/Moon/Venus references are frozen as a product convention. They are supported by the Drik Panchang Manglik calculator documentation, which explicitly checks Lagna, Moon, and Venus and includes both the first and second houses alongside the fourth, seventh, eighth, and twelfth.
