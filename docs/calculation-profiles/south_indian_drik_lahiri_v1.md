# South Indian Drik Lahiri v1

This profile is immutable once released.

## Astronomy

- Zodiac: sidereal
- Ayanamsha: Lahiri
- Planet positions: geocentric apparent positions returned by Swiss Ephemeris
- Lunar node: true node
- Ketu: point exactly 180 degrees opposite Rahu
- Houses: whole sign
- Birth-time input: naive local civil time plus an IANA timezone
- Ambiguous civil times: caller must provide `fold` as `0` or `1`
- Nonexistent civil times: rejected

## Returned bodies

Sun, Moon, Mars, Mercury, Jupiter, Venus, Saturn, Rahu, and Ketu.

## Versioning

Corrections that alter a calculation result require a new profile ID. Existing profile
behavior must not be changed silently.
